# Kerala Eco-Tourism Website

A responsive tourism website showcasing Kerala’s breathtaking destinations, cultural heritage, eco-tourism experiences, and traditional art forms through an immersive frontend design.

---

## Features

- Responsive navigation bar
- Interactive hero carousel
- Eco-tourism destination showcase
- Kerala cultural experiences section
- Responsive card layouts
- Footer with contact and social links
- Mobile-friendly interface refinements
- Organized asset structure

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
